Installation

Folge diesen Schritten, um die Globify Desktop App zu installieren:

  1. Entpacke die heruntergeladene ZIP-Datei
  2. Öffne ein Terminal-/Kommandozeilenfenster
  3. Navigiere zum entpackten Verzeichnis mit cd Pfad/zum/Globify-desktop
  4. Installiere die Abhängigkeiten mit npm install
  5. Starte die App mit npm start
Hinweis: Du benötigst Node.js und npm auf deinem Computer, um die App auszuführen. Falls du diese noch nicht installiert hast, kannst du sie von nodejs.org herunterladen.

Systemanforderungen

  • Windows, macOS oder Linux
  • Node.js (Version 14 oder höher)
  • 4 GB RAM empfohlen
  • 100 MB freier Festplattenspeicher
